# Termux Banner
#Script by Parixit Sutariya

Installation Step :

1) git clone https://github.com/Bhai4You/Edris-Banner


2) cd Edris-Banner


3) chmod +x requirement.sh


4) chmod +x t-ban.sh


5) bash requirement.sh


6) bash t-ban.sh



Uninstallation Step : (Remove Logo From Termux)

1) bash remove.sh

2) Done !!!
